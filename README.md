## How to Use
1. Click "Record" (or press 'A') to start recording
2. Click "Stop" (or press 'S') when finished
3. Click "Save to Drive" (or press 'D') to save
4. Click "Delete & Redo" (or press 'F') to start over

## Keyboard Shortcuts
- **A** - Start Recording
- **S** - Stop Recording  
- **D** - Save to Drive
- **F** - Delete & Redo
- **G** - Connect/Disconnect Google Drive
